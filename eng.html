ABOUT ME

I am a programmer with a sincere passion and it's hard for me to imagine life without programming even for myself.

I started actively learning programming in April 2022, taking Java courses, although I had some programming experience. Currently, I am focusing on polishing my skills in using raw HTML, CSS and JavaScript languages, but I am also starting learning back-end technology to be able to better understand how the application works as broadly as possible. With July, I'm going to start using frameworks such as Angular or Bootstrap.
